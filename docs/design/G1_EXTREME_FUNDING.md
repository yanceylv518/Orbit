# G1 极端 Funding 反应信号预注册

预注册时间：2026-07-13  
状态：训练搜索已完成，G1 FAIL；未创建或打开新锁箱

## 1. 假设

Funding 极正表示多头拥挤并向空头付费，Funding 极负表示空头拥挤并向多头付费。候选信号在结算时点 Funding 相对其历史分布达到极端时，逆着拥挤方向持有单腿 perp：

```text
funding_rate > 0 -> short perp
funding_rate < 0 -> long perp
gross_return_pct = -sign(funding_rate) * (exit_price / entry_price - 1) * 100
net_return_pct = gross_return_pct - 0.14%
```

这与无条件锚点回归和 Funding Carry 属于不同信号类别，但仍必须先证明成本后 alpha，再讨论执行。

## 2. 无未来泄漏定义

- 当前 Funding 是否极端，只能使用当前结算点之前的 Funding 绝对值历史计算阈值；当前值不进入自己的分位数阈值；
- Funding 在结算时点视为已知，入场价格取该时点之后第一个 15m 已收盘 K 线的 close；
- 出场取入场后固定 `H` 根 15m K 线的 close；
- 持仓期间忽略其他 Funding 信号，出场后才允许下一事件，保证事件不重叠；
- 尾部不足完整持有期的事件丢弃，不以未来价格补齐。

## 3. 训练搜索网格

训练数据使用已被前序研究打开、已不再具备新锁箱资格的 BTCUSDT、ETHUSDT、BNBUSDT、SOLUSDT 15m perp OHLC 与对应 Funding。

冻结网格：

- Funding lookback：`90 / 180 / 360` 次结算；
- 极端阈值：历史 `|funding_rate|` 的 `90% / 95% / 97.5%` 分位数；
- 持有期：`4 / 16 / 32 / 96` 根 15m K 线，即 `1h / 4h / 8h / 24h`；
- 共 `3 x 3 x 4 = 36` 个参数组合；
- 成本：单腿 perp 往返 `2 x (0.05% taker + 0.02% slippage) = 0.14%`；不允许 maker 优化。

Bootstrap 固定 `10,000` 次、种子 `20260713`，输出均值 95% 区间；同时输出胜率 Wilson 95% 区间。

## 4. 训练候选选择门

单市场合格必须同时满足：

1. 非重叠事件数 `>= 30`；
2. 成本后平均收益 `> 0`；
3. Bootstrap 均值 95% 下界 `> 0`。

参数组合进入候选池必须满足：

1. 至少 `3/4` 市场单市场合格；
2. 四市场合并平均净收益 `> 0`；
3. 合并 bootstrap 均值 95% 下界 `> 0`。

若候选池非空，按以下冻结顺序选择唯一候选：组合 bootstrap 下界最高，其次平均净收益最高、事件数最多、lookback 更长、分位数更高、持有期更短。若候选池为空，**G1 在训练阶段直接 FAIL，不创建或打开新锁箱**。

## 5. 条件触发的锁箱

只有训练门 PASS 才允许获取从未参与选择的新市场 15m OHLC+Funding，先冻结文件哈希，再用唯一参数一次性判定。锁箱单市场仍要求 `>=30` 事件、正平均收益、正 bootstrap 下界；组合要求盈利市场过半且组合均值/下界为正。

锁箱结果不得用于修改 lookback、分位数、持有期、成本、市场或入场对齐规则。

- 单元测试锁定方向、成本、非重叠、价格对齐、历史阈值排除当前值和统计门；

## 7. 训练结果（2026-07-13）

按上述冻结协议运行 `36` 个参数组合，使用 BTCUSDT、ETHUSDT、BNBUSDT、SOLUSDT 共四个已打开市场。完整机器可读结果保存在本地忽略文件 `var/calibration/g1_extreme_funding_training.json`。

- 候选组合：`0 / 36`；所有组合均为 `0 / 4` 单市场合格；
- 按预注册排序最接近门槛的诊断组合：`lookback=360`、`quantile=95%`、`holding=1h`；
- 该组合合并 `135` 个非重叠事件，平均净收益 `-0.0665%`，bootstrap 95% 区间 `[-0.2032%, +0.0953%]`；
- 该组合单市场结果：BTC `31` 事件、均值 `-0.2225%`、下界 `-0.3903%`；ETH `33`、`-0.0146%`、`-0.3958%`；BNB `32`、`-0.1639%`、`-0.3133%`；SOL `39`、`+0.0934%`、`-0.1691%`；
- 全网格最高合并均值来自 `lookback=180`、`quantile=90%`、`holding=24h`：`+0.2272%`，但 bootstrap 下界为 `-0.3429%`，且仍为 `0 / 4` 单市场合格。

**结论：G1 在训练阶段 FAIL。** 极端 Funding 后逆拥挤方向的单腿反应，在冻结的 `0.14%` 往返成本和样本稳健性门槛下没有可冻结候选。按预注册规则，不获取新市场、不打开锁箱，也不根据本结果回调参数。该结果只否定当前定义和成本口径下的 G1，不等同于否定所有 Funding 条件信号。
