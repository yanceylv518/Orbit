## 1. 研究问题

TB3 PASS 依赖单一 `mom28 / vol28 / 7 天再平衡`。由于 `28` 是 TB1 动量网格 `{28,84,168}` 的最短边界，且单挑一个参数组合存在幸运数字风险，TB-R 在 TB4 纸面前向前做最后一次历史稳健性诊断：

1. `28/7` 周围是否是连续可用区域，而不是孤立尖点；
2. 更短周期是否因反应更快而改善，还是因噪声和换手恶化；
3. 不挑单一赢家的固定多周期集成，能否通过 TB3 原准入门。

本轮历史已被前序研究检视，证据等级仍为 `BACKTEST_CONFIRMATION`。Part A 只诊断，不得选择部署参数；Part B 是跑数前冻结的唯一候选。

### 3.1 冻结平滑性判据

中央格点为 `(lookback=28, rebalance=7)`。四个轴向邻居固定为：

`(14,7), (56,7), (28,3), (28,14)`。

单个邻居只有同时满足以下“支持门”才算 supportive：

1. 聚合总净收益 `> 0`；

6. 正滚动 12 个月占比 `>= 50%`；

曲面判定为 `SMOOTH` 必须全部满足：

1. 中央 `28/7` 自身通过完整 TB3 门；
2. 四个轴向邻居至少 `3/4` supportive；

4. 中央聚合年化净收益、Calmar、Sortino分别不超过四个轴向邻居对应指标中位数的 `2.0` 倍。

任一项失败即为 `SPIKE_OR_BOUNDARY_RISK`。这个判据只回答稳健性，不授予某个邻居部署资格。

## 4. Part B：固定多周期集成

唯一候选固定使用全部 lookback：

`14 / 28 / 56 / 84 / 168` 天。

每个市场在信号收盘 `t` 计算：

```text
component_j(t) = sign(close(t) / close(t - lookback_j) - 1)
ensemble_signal(t) = mean(component_j(t) for j in fixed_lookbacks)
raw_weight_i(t) = ensemble_signal_i(t) * 10% / (sqrt(active_markets) * annual_vol_28_i(t))
```

五个分量各占 `20%`，不按历史表现加权。通常每个分量为 `-1/+1`，若某 lookback 起终价格恰好相等则该分量为 `0`；因此 `ensemble_signal` 理论取值为 `-1` 到 `1`、步长 `0.2`，包含 `0`。信号幅度直接缩放风险权重，完全抵消时目标权重为 0，分歧越大仓位越小。组合按 7 天再平衡，其余数据、成本和执行规则与第 2 节一致。

该候选没有训练选择、没有可调 lookback、没有阈值或二次权重。不得根据 Part A 曲面删除表现较差的周期。
